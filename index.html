Domain 4 · 20% · Claude API

Prompt Engineering & Structured Output

Stable API ground — high confidence. Maps to CI/CD and Structured Data Extraction scenarios.

Nail these exam points tick as mastered

📦 Batch vs. Sync — memorize the line
Message Batches API: 50% cheaper · up to 24h · no latency SLA · no multi-turn tool calling · custom_id to correlate. → Overnight / weekly work.
Sync API: blocking pre-merge checks where you need the answer now.
✅ The schema-validity nuance
tool_use + JSON schema eliminates syntax errors — not semantic ones (totals that don’t sum). Retry-with-error-feedback fixes structural errors, but cannot conjure info that’s simply absent from the source.
